Bienvenido al Control General UCP

Utilice el menú lateral () para acceder a los módulos de la aplicación. Haga clic en para volver aquí.

REGISTRO DE INGRESO

Módulo principal para registrar el acceso de personal, móviles, visitas, proveedores y otros. Permite la carga manual y el escaneo de códigos/QR para asociar la persona a un perfil y registrar la hora de ingreso.

REPORTES

Permite visualizar el listado de ingresos registrados, aplicar filtros por fecha, código o dominio, y exportar la información resultante a formatos XLS (Excel) o PDF.

CARGA DE BASES (Excel)

Módulo para actualizar las bases de datos internas (Dotación y Pases Body Scan) a través de archivos Excel. Es esencial para la correcta identificación de los códigos escaneados.
